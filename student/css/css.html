    # css

    # 1、字体的属性

    # 1.1 color:字段颜色

    ###色值：十六进制、RGB、rgba、颜色名。

    # color: #ff0000; 红色

    # 颜色使用：背景色、边框色等。

    # 1.2 font-family:字体

    # 中文：微软雅黑、宋体

    # 英文：Arial、Consola

    # font-family:"Arial","Consola"

    # 1.3 font-size:字号

    # 文字大小 。自己设置一个通用字体大小，如果不设置，页面会有一个默认的字段大小。

    ####谷歌浏览器：最小是显示12Px。如果文字设置小于12像素，会自动显示成12Px.

    # IE浏览器：可以随意设置，甚至是1px

    # 常用字号像素值：12/14/16/18/22

    # 1.4 line-height:行高

    # 一行文字实际占有的高度。文字在自己的行高内，垂直居中。

    ####单 位：像素px或百分比

    # 注意：行高属性在书写，必须写在字号的后面。

    # 行高、字号、字体都是font字段的单独属性

    # 特殊用途：要让一行文字在盒子里垂直居中，让行高等于盒子高。只能单行文字的垂直居中

    <style type="text/css">		
	p{
		height: 100px;
		width: 300px;
		border: 1px solid red;
		color: #252525;
		font: 14px/100px "微软雅黑";
		
	}
</style>

<body>
	<div class="box1" id="box1">
		<p>看看我的颜色听谁的</p>
	</div>
</body>

    # font: 14px/27px "宋体";

    # 1.5 font-weight: 字体的加粗

    ####属性值：数字、单词

    # 数字：100，200......700

    # 两个单词： bold =700加粗，normal =400 正常。

    # font: bold 14px/40px "宋体" 复合属性写法

    # 1.6 font-style:字段的样式

    # normal:正常 italic:斜体-会找一个有斜体的字体进行替换 oblique:斜体--正常的展示斜体

    # font: italic boid 14px/22px "宋体"; 复合属性书写

    # 2、文本

    ###2.1 text-align:对齐

    # 对齐方式：水平左对齐、水平右对齐、水平居中。 默认文本left左对齐

    # 对应的属性值：left、right、center

    ###2.2 text-indent: 文本缩进

    # em 或者px像素或者百度比为单位

    # 如果是百分比，参考的不是本身的宽度，而是父盒子宽度的百分之多少。

    ###2.3 text-decoration：文本修饰

    # 对于文本整体样式的一个修饰效果。

    # 有四种常见样式：

    # 正常，没有修饰：none;

    # 下划线：underline;

    # 中划线，删除线：line-through

    ####上划线：overline

    ####所有的文本除了（a）标签，其他都默认为none;

    # 3、浮动 float

    # 对应属性值 ： left、right、center

    # 因为浮动会脱离文档流，导致底部内容顶上，所以需要清除浮动

    # 消除浮动的方法

    ##### 3.1  额外标签法（在最后一个浮动标签后，新加一个标签，给其设置clear：both；）（不推荐） 会增加一个DOM元素 

	div {
		width: 100px;
		height: 200px;
		border: 1px solid red;
	}
	
	.box1 {
		float: left;
	}
	
	.box2 {
		width: 50px;
		height: 50px;
		background: pink;
	}
	.clear{
		clear: both;
	}
</style>

<body>
	<div class="box1">
	</div>
	<p class="clear"></p>
	<div class="box2">
	</div>
</body>
    # 3.2 父级添加overflow属性（父元素添加overflow:hidden）（不推荐）如果我们清除了浮动，父元素自动检测子盒子最高的高度，然后与其同高。

		float: left;
	}
	
	.box2 {
		width: 50px;
		height: 50px;
		background: pink;
	}
	.box {
		overflow: hidden;
	}
</style>

        <div class="box1">
        </div>

    <div class="box2">
	</div>

</body>
    # 3.3 使用after伪元素清除浮动（推荐使用） （注意：给父元素设置）
    <style type="text/css">
div {

		border: 1px solid red;

}
.box1 {
	width: 200px;
	height: 100px;
	float: left;
}

.box2 {
	width: 50px;
	height: 50px;
	background: pink;
}

.clearfix:after {
	content: "";
	display: block;
	height: 0;
	clear: both;
	visibility: hidden;
}

.clearfix {
	/*IE兼容设置*/
	*zoom: 1;
}
</style>

<body>

	<div class="clearfix">

		<div class="box1 ">

		</div>
	</div>
	<div class="box2 ">

	</div>
</body>

    # 3.4 使用before和after双伪元素清除浮动

    <style type="text/css">

div {

		border: 1px solid red;

}
.box1 {
	width: 200px;
	height: 100px;
	float: left;
}

.box2 {
	width: 50px;
	height: 50px;
	background: pink;
}

.clearfix:after,.clearfix:before{
	content: "";
	display: table;
}
.clearfix:after{
	clear: both;
}

.clearfix {
	/*IE兼容设置*/
	*zoom: 1;
}
</style>

<body>

	<div class="clearfix">

		<div class="box1 ">

		</div>

	</div>
<div class="box2 ">

</div>
</body>

    # 4:margin 塌陷

    # 4.1 😗*垂直之间塌陷的原则是以两盒子最大的外边距为准。 **  我们肯定会很理所当然的认定这两个盒子之间的距离为90px，事实上并非如此.

    <style type="text/css">

.box1,.box2{
	width: 100px;
	height: 100px;
	border: 1px solid red;
}

.box1{
	height: 130px;
	margin-bottom: 50px;  (以当前为准)
}
.box2{
	margin-top: 40px;
}
</style>

	<body>
		<div class="box">
		<div class="box1">
		</div>
		<div class="box2 ">
		</div>
	</div>

</body>

    # 4.2: 嵌套关系(父级元素塌陷) 　　　 子盒子和父盒子之间并没出现期望的120px间隙而是父盒子与子盒子一起与页面顶端产生了120px间隙（即父级盒子往下塌陷了120px）。

    <style type="text/css">

*{
	margin: 0;
	padding: 0;
}
.box1,.box{
	width: 300px;
	height: 200px;
	/*border: 1px solid red;*/
	background: blue;
}
.box1{
	width: 100px;
	height: 130px;
	background: pink;
	margin-top: 120px;
}
</style>

	<body>
	<div class="box">
		<div class="box1">
		</div>
	</div>

</body>

    # 4.1.3 为父盒子设定padding值；
    # 4.1.4为父盒子添加position：fixed；
    # 4.1.5 为父盒子添加 display：table；（会改变盒子的大小，table属性会列表化）

    # 5: Position 定位

    # 有左就以左为原点，有右就以右为原点

    • #####static - 布局正常。 这是默认值。

    • #####relative - 相对于其正常位置定位。 (以自身为原点) 相对定位 （原来的位置会保存着）

    • #####absolute - 相对于其具有非静态位置值的其第一祖先定位。 一层一层住外找，如果没有就以它为基准 （绝对定位 ）（脱离文档流了）

    • #####fixed - 相对于浏览器窗口定位。（固定定位）
